# IIITDM Kurnool Campus Navigation System

## 1. Project Title
Interactive Campus Navigation System for IIITDM Kurnool

## 3. Problem Statement
New students, visitors, and parents often find it difficult to navigate the IIITDM Kurnool campus due to its large area and limited signboards.

## 4. Proposed Solution
We propose a web-based interactive campus navigation system that displays the IIITDM Kurnool campus map along with important locations such as academic blocks, hostels, and other facilities.

## 5. Tools and Technologies Used
- HTML for structure
- CSS for styling
- JavaScript for logic
- Leaflet.js for interactive maps
- OpenStreetMap for map tiles
- GitHub for version control and hosting

## 6. Current Progress
- Project repository created on GitHub
- Website hosted using GitHub Pages
- Campus map successfully displayed
- Correct campus location marked on the map

## 7. Future Scope
- Add markers for hostels, academic blocks, and library
- Add search functionality for buildings
- Add walking routes between locations
- Improve user interface and design

## 8. Conclusion
This project aims to simplify campus navigation and provide an easy-to-use digital map for IIITDM Kurnool students and visitors.
